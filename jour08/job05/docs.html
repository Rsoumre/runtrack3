Introduction

RuntrackJS est une collection d'exercices et d'exemples pour apprendre le développement web moderne. Dans cette documentation, vous trouverez tous les concepts nécessaires pour maîtriser JavaScript et Tailwind CSS.

Pro tip: Commencez par les bases avant de passer aux concepts avancés.

Installation

Pour commencer avec Tailwind CSS, vous pouvez utiliser le CDN :

<script src="https://cdn.tailwindcss.com"></script>

Ou l'installer via npm :

npm install tailwindcss

Utilisation

Classes utilitaires

  • Flexbox : flex, items-center
  • Grille : grid, grid-cols-2
  • Espacement : p-4, m-2, gap-6

Responsive Design

  • Mobile : text-sm
  • Tablette : md:text-base
  • Desktop : lg:text-lg
